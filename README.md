Project Name: SmartEdu Assistant

Description:
SmartEdu Assistant is an AI-based university course counselling chatbot developed using Python, Tkinter, and SQLite.

Main Features:
- Text-based chatbot interface
- NLP-based intent matching
- Course fee, duration, requirements, and description answers
- SQLite knowledge base
- Unknown question saving
- Admin panel
- Admin-approved answer adding
- Retrain Knowledge Model feature
- Chat history saving

How to Run:
1. Install Python.
2. Open the project folder.
3. Run database.py first if smartedu.db is not available.
4. Run app.py.

Command:
python app.py
